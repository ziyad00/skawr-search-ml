# SKAWR Search ML

Custom transformer model for general domain marketplace search engine.

## Overview

This project develops a custom transformer model to improve search relevance across multiple domains (automotive, electronics, fashion, etc.) for SKAWR marketplace search.

## Problem Statement

Current search using pre-trained embeddings returns semantically similar but irrelevant results. For example, "sports car" queries return generic automotive items rather than actual sports cars.

## Solution

Build a custom transformer model trained specifically for marketplace search across multiple domains.

## Project Structure

```
├── data/               # Training data collection
├── models/             # Model architecture and training
├── evaluation/         # Model evaluation and testing
├── integration/        # Integration with SKAWR search system
└── docs/              # Documentation and research
```
